Section // Basic usage
======================

Test case #0: Basic punctuation formatting

    >>> text_indentation("Hello. World?")

    <BLANKLINE>

    <BLANKLINE>

    <BLANKLINE>
    Yes?

Test case #2: Multiple punctuation marks in long text

    >>> text_indentation("A? B. C: D")

    B.
    <BLANKLINE>
    C:
    <BLANKLINE>
    D
