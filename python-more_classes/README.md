# python-more_classes

Solutions for the "More Classes" Holberton/ALX Python project.

## Files
- `0-rectangle.py` - empty Rectangle class
- `1-rectangle.py` - Rectangle with private width/height and validation
- `2-rectangle.py` - Rectangle with `area()` and `perimeter()`
- `3-rectangle.py` - `__str__` printing the rectangle using `#`
- `4-rectangle.py` - `__repr__` that recreates the object
- `5-rectangle.py` - `__del__` printing "Bye rectangle..."
- `6-rectangle.py` - class attribute `number_of_instances`
- `7-rectangle.py` - class attribute `print_symbol` used in `__str__`
- `8-rectangle.py` - static method `bigger_or_equal(rect_1, rect_2)`
- `9-rectangle.py` - classmethod `square(size)`

## Requirements
- All files must be executable and start with `#!/usr/bin/python3`
- Use Python 3.8.5 on Ubuntu 20.04 LTS
- Follow `pycodestyle` (version 2.7.*)

## Running tests
Each task includes a `N-main.py` script to show expected behavior
